Dynamic ExamplesÔÉÅ

The theme supports the dynamic update of links used in the step-by-step instructions by adding a query suffix to documentation page URL:

  • ?live=instance

  • ?domain=example.net

  • unchanged (no query suffix)

Example http://localhost:8080/geoserver links:

  • http://localhost:8080/geoserver/web

  • WMS 1.3.0 GetCapabilites

Example http://localhost:8080/geonetwork links:

  • http://localhost:8080/geonetwork/srv

  • CSW GetCapabilities
