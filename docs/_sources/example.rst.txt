****************
Dynamic Examples
****************

The theme supports the dynamic update of links used in the step-by-step instructions by adding a query suffix to documentation page URL:

* `?live=instance <example.html?live=instance>`__
* `?domain=example.net <example.html?domain=example.net>`__
* `unchanged (no query suffix) <example.html>`__


Example ``http://localhost:8080/geoserver`` links:

* http://localhost:8080/geoserver/web
* `WMS 1.3.0 GetCapabilites <http://localhost:8080/geoserver/ows?service=WMS&version=1.3.0&request=GetCapabilities>`__

Example ``http://localhost:8080/geonetwork`` links:

* http://localhost:8080/geonetwork/srv
* `CSW GetCapabilities <http://localhost:8080/geonetwork/srv/eng/csw>`__
